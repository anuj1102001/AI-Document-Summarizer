# AI Document Summarizer

AI-powered document summarization API built using FastAPI and Google's Gemini API.

## Features

- 📄 Upload PDF documents
- 🤖 AI-generated summaries using Gemini
- 📝 Multiple summary styles
- ⚡ FastAPI REST API
